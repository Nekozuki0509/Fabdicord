#### Fabdicord
## 説明
[Velodicord](https://modrinth.com/plugin/velodicord)のfabricサーバー側のアドオン
## 使い方
1. fabricサーバーのmodssフォルダにこのmodを入れて再起動
1. config/fabdicordconfig.propertiesを編集
1. 楽しみましょう!
## configファイル
```
SERVER_NAME=s #String | default: s | comment: it's this server name
```
